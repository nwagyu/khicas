5.47.2  Inverse of a matrix: inv /

The inv command finds the inverse of a matrix.

  • inv takes one argument:
    A, a matrix.
  • inv(A) returns the inverse matrix of A.

Note that 1/A is another way to find the inverse of a matrix.

Example

Input:

inv([[1,2],[3,4]])

or:

1/[[1,2],[3,4]]

or:

A:=[[1,2],[3,4]];1/A
