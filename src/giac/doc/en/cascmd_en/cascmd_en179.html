5.12.1  Evaluating an expression: eval

The eval command is used to evaluate an expression. Since Xcas always evaluates expressions entered in the command line, eval is mainly used to evaluate a sub-expression in the expression editor (see Section 3.3).

Examples

  • Input:
    a:=2
    Output:
      2
  • Input:
    eval(2+3*a)
    or:
    2+3*a
    Output:
      8
