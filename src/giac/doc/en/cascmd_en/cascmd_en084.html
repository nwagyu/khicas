5.3.8  ASCII code of a string: asc

The asc command finds the ASCII codes of all the characters in a string.

  • asc takes one argument:
    str, a string.
  • asc(str) returns the list of the ASCII codes of the characters of s.

Examples

  • Input:
    asc("abcd")
    Output:
      ⎡
    ⎣    		97,98,99,100⎤
    ⎦
  • Input:
    asc("a")
    Output:
      ⎡
    ⎣    		97⎤
    ⎦
