5.3.13  Transforming a real number into a string: cat +

The cat command (see Section 5.3.11) can also be used to transform a real number into a string, as can + (see Section 5.3.12).

If cat has a real number as an argument, the result will be a string.

Example

Input:

cat(123)

Output:

"123"


Similarly, if you add a real number to an empty string, the result will be a string.

Example

Input:

""+123

Output:

"123"
