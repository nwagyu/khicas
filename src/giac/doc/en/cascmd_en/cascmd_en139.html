5.8.4  nth root: root

The root command finds roots of numbers.

  • root takes two arguments:
    n and a, numbers.
  • root(n,a) returns the nth root of a (i.e. a1/n). If a<0, the nth root is a complex number with argument 2π/n.

Examples

  • Input:
    root(3,2)
    Output:
      2
    1
    3
     
  • Input:
    root(3,2.0)
    Output:
      1.25992104989
  • Input:
    root(3,sqrt(2))
    Output:
      2
    1
    6
