5.16.5  Repeated function composition: @@

With Xcas, the repeated composition of a function with itself n ∈ ℕ times is done with the infixed operator @@ (see Section 5.14.1).

Examples

  • Input:
    (sin@@3)(x)
    Output:
      sin⎛
    ⎝    		sin⎛
    ⎝    		sinx⎞
    ⎠    		⎞
    ⎠
  • Input:
    (sin@@2)(pi/2)
    Output:
      sin⎛
    ⎝    		1⎞
    ⎠
