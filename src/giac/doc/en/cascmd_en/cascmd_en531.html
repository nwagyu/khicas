5.49.4  LQ decomposition (HP compatible): LQ

The LQ decomposition of a matrix A is A=LQP, where L is lower triangular the same size as A (if A is not square, then ℓi,j=0 for i>j), Q is an orthogonal matrix, and P is a permutation matrix. The LQ command finds the LQ decomposition of a matrix.

  • LQ takes one argument:
    A, a matrix.
  • LQ(A) returns a list [L,Q,P] of the matrices given by the LQ decomposition.
