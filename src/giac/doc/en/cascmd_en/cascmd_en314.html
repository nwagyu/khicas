5.27.26  Random polynomial: randpoly randPoly

The randpoly command finds random polynomials.
randPoly is a synonym for randpoly.

  • randpoly takes two optional arguments:
    • Optionally x, the name of a variable (by default x).
    • Optionally n, an integer (by default 10).
    The order of the arguments is not important.
  • randpoly(⟨ x⟩ ⟨ ,n⟩) returns a monic polynomial in the variable x of degree n, having as coefficients random integers evenly distributed on -99..+99.

Examples

  • Input:
    randpoly(t,4)
    Output (for example):
      t4+86 t3−97 t2−82 t+7
  • Input:
    randpoly(4)
    Output (for example):
      x4−27 x3+26 x2−89 x+63
  • Input:
    randpoly(4,u)
    Output (for example):
      u4−49 u3−86 u2−64 u−30
