Examples

  • Input (in real mode):
    fsolve(x^3-1,x)
    Output:
    1.0
  • Input (in complex mode):
    fsolve(x^3-1,x)
    Output:
    ⎡
    ⎣    		−0.5−0.866025403784i,−0.5+0.866025403784i,1.0⎤
    ⎦
  • Input (in any mode):
    cfsolve(x^3-1,x)
    Output:
    ⎡
    ⎣    		−0.5−0.866025403784 i,−0.5+0.866025403784 i,1.0⎤
    ⎦
  • Input:
    fsolve(sin(x)=2)
    Output:
    ⎡
    ⎣    		⎤
    ⎦
  • Input:
    cfsolve(sin(x)=2)
    Output:
    ⎡
    ⎣    		1.57079632679−1.31695789692 i,1.57079632679+1.31695789692 i⎤
    ⎦
