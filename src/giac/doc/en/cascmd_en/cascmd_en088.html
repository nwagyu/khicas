5.3.12  Adding an object to a string: +

The ’+’ command can be used like cat (see Section 5.3.11), and the + operator is the infixed version. (See Section 5.16.1 for other uses of + and ’+’.)

  • ’+’ takes one argument:
    seq, a sequence of objects, at least one of which is a string.
  • ’+’(seq) returns the concatenation of the string representations of the objects in seq.


Warning.
+ is infixed and ’+’ is prefixed.

Examples

  • Input:
    ’+’("abcd",3,"d")
    or:
    "abcd"+3+"d"
    Output:
      "abcd3d"
  • Input:
    c:=5
    then:
    "abcd"+c+"d"
    or:
    ’+’("abcd",c,"d")
    Output:
      "abcd5d"
