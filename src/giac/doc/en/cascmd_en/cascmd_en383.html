5.34.16  Inverse of a matrix with coefficients in ℤ/pℤ: inv inverse

The inv command can find the inverse of a matrix with elements in ℤ/pℤ. (See also Section 5.47.2.)
inverse is a synonym for inv.

  • inv takes one argument:
    A, a matrix in ℤ/pℤ.
  • inv(A) returns the inverse of the matrix A.

Example

Input:

inv([[1,2,9]%13,[3,10,0]%13,[3,11,1]%13])

or:

inverse([[1,2,9]%13,[3,10,0]%13,[3,11,1]%13])

or:

inv([[1,2,9],[3,10,0],[3,11,1]]%13)

or:

inverse([[1,2,9],[3,10,0],[3,11,1]]%13)
